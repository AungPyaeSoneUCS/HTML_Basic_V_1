
# HTML Basic Project (v1.0)

A foundational web development project focusing on the essential building blocks of the modern web. This repository serves as a reference for mastering structural HTML5 tags, document flow, and the integration of basic web elements.

## 🚀 Overview

The **HTML_Basic_V_1** project is designed to demonstrate proficiency in creating clean, semantic, and well-structured web pages. It covers the fundamental principles of document structure, form handling, and content layout, providing a stable base for future styling and interactivity.

## 🛠 Core Concepts Covered

* **Semantic HTML5:** Using structural elements like `<header>`, `<nav>`, `<main>`, and `<footer>` for better accessibility and SEO.
* **Form Elements:** Implementation of inputs, text areas, and submission handling for user data.
* **Media Integration:** Proper embedding of images, links, and multimedia content.
* **Document Structure:** Proper head metadata, viewport optimization, and charset implementation.

## 📋 Project Structure

* `index.html`: The main entry point featuring the primary page layout.
* `assets/`: Folder containing localized images, CSS (if applicable), and scripts.
* `pages/`: Supplementary HTML files demonstrating secondary layouts or features.

## 💻 How to View

1. **Clone the repository:**
```bash
git clone https://github.com/AungPyaeSoneUCS/HTML_Basic_V_1.git

```


2. **Open in Browser:** Navigate to the folder and open `index.html` in any modern web browser (Chrome, Firefox, Edge, etc.).
3. **Local Development:** You can use a live server extension (like "Live Server" for VS Code) to view changes in real-time.
